# 🛍️ MiniStore - Modern E-Commerce Mobile App

MiniStore ek high-end, premium UI/UX design wali React Native application hai jo modern shopping experience provide karti hai. Is mein **MERN Stack** (Backend) aur **React Native** (Frontend) ka istemal kiya gaya hai.

## ✨ Features
- 🚀 **Advanced UI/UX:** Designer-level premium interface.
- 🔐 **Secure Authentication:** JWT based login/signup system.
- 🛒 **Cart Management:** Full cart functionality (Add, Remove, Quantity Update).
- 📦 **Order System:** Real-time order placement with backend integration.
- 🎨 **Modern Design:** Soft shadows, premium icons, and smooth transitions.
- 📱 **Responsive:** Android aur iOS dono ke liye optimized.

## 🛠️ Tech Stack
- **Frontend:** React Native (Expo)
- **State Management:** React Context API
- **Icons:** Expo Vector Icons (Ionicons)
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Networking:** Axios
